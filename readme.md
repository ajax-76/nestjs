

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript  repository.

## Installation

```
install docker and docker compose
```

## Running the app 
```
Go to root and run the following command
```

```bash
# development
$ docker compose build

# watch mode
$ docker compose up -d 

```

## Test

```bash
# unit tests
$ npm run test
```
